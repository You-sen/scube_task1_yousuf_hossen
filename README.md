## Getting Started

HOW to run this app???
Option 1 ->
 ->Create a flutter project
 ->download/get assets and lib folder
 ->get pubspec.yaml
 ->change project  at the top in pubspec.yaml with your folder name
 ->change 2nd import path name at the top of main.dart you folder name
->Allow vs to get all the dependency and you are ready to go

Option 2 ->
 ->Clone/download the whole repo
 ->get all the dependency and you are good to go
